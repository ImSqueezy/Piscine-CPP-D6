# Description
CPP Module 06 follows the series of CPP Modules. This one focuses on type conversions and simple parsers.

# Instructions
This project follows the basic compilation of C/C++!

## Compilation
Make within each exercise and execute each's corresponding program.

## ex00/
The first exercice Introduces static methods and type conversions. It is required write a program that takes an argument, parse it and convert it from char, int, float to double.

Usage:
```bash
./a.out arg
```

where arg is a string representing a int, float, double or a single character value; parsing is needed.

## ex01/
This second exercice is about making use of reinterpret_cast and understand the concept of serialization and deserialization.

Usage:
```bash
/a.out
```

The output demonstrates the use of reinterpret_cast, casting from Data to uintptr_t (large data type fit for any data type conversion) and viseversa.
